# @atomic/ui

An atomic, CSS-first, user interface toolkit and library for building custom elements and user interfaces. Built on the web platform and the [Lit](https://lit.dev/) library.

## Installation

```bash
pnpm add @atomic/ui
```

### Testing

`vitest` + storybook interaction tests (for ui library).

For '@atomic/ui-site' `@testing-library/react`.

- [`vitest`](https://vitest.dev/)
- [`@storybook/addon-vitest`](https://storybook.js.org/docs/writing-tests/integrations/vitest-addon)
- [`@testing-library/react`](https://testing-library.com/docs/react-testing-library/intro/)
